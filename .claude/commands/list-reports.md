Liệt kê tất cả report files có sẵn với metadata.

Việc cần làm:
1. Scan reports/ directory cho tất cả .md files
2. Đọc metadata cho mỗi file:
   - Filename
   - Size (KB)
   - Last modified date/time
   - First line (title/description)
3. Categorize reports:
   - Assessment reports (assessment-report-*.md)
   - Test results (test-results.md, single-api-test-results.md)
   - Analysis reports (blacklist-summary.md, test-rules-summary.md)
   - Generated by /report command (bugs-found.md, coverage-matrix.md, ai-summary.md)
4. Display table format:
